03

⚙️Tech Engine Hub — GitHub Setup

GitHub is your public proof-of-work engine. Follow each step to establish your technical foundation.

  • Create or log into your GitHub account at github.com
  • Create a new repository named for your chosen passion project (e.g., indie-game-studio-journey)
  • Enable GitHub Pages in repo Settings → Pages → Deploy from branch (main)
  • Upload your HTML to a named subfolder in the /docs folder in the repo — this becomes your live page
  • GitHub Editor QuirkSubfolders are created in the file creation dialogue with a following slash
  • Save your HTML also in Google Docs or Google Keep for platform portability
  • For longer HTML, on a copy and paste, be sure all the HTML was copied, right down to the close HTML tag
  • If HTML won't render in a browser, use Claude to debug, where again copy and paste is your friend
  • Once confidence in GitHub is established, use of Google to "save your work" can become more judicious
🔗 Note: The Project Engine browser utility link will be updated once available. For now, navigate GitHub directly via your browser.
