Phase I: IP & Infrastructure Setup (25% Weight)

1. GitHub Infrastructure Setup

Credit: 10.0% (Founder)

**Actionable Subtasks:**

  • Create and secure private GitHub Organization and core repository.
  • Implement **2FA enforcement** for all team members.
  • Define and implement **Branch Protection Rules** for `main` branch.
  • Enable **Secret Scanning** and **Dependabot** on the repository.

**Definition of Done (DoD):** Security Audit log confirms 2FA enforcement and Secret Scanning is active. All security files (`SECURITY.md`, `CODE_OF_CONDUCT.md`) are committed to `main`.

2. Legal/Agreement Setup

Credit: 5.0% (Founder)

**Actionable Subtasks:**

  • Finalize and execute **NDA** with all partners.
  • Draft and finalize **Contributor License Agreement (CLA)** for IP assignment.
  • Define and draft smart contract parameters for automated contribution rewards.

**Definition of Done (DoD):** Fully signed NDA and CLA documents are secured for all named contributors (Founder, Market, Tech).

3. Initial Tech Review

Credit: 10.0% (Technical Assistant)

**Actionable Subtasks:**

  • Select core technology stack (e.g., Python/Django, Node/Express, Solidity/EVM).
  • Define high-level architecture diagram (e.g., database, hosting, CI/CD pipeline).
  • Create and document a basic **development environment setup** guide.

**Definition of Done (DoD):** Architecture document and toolchain selection are approved by the Founder. Local development environment is verified working.

Phase II: Core Product Development (40% Weight)

4. Backend Setup (API, Database, Logic)

Credit: 15.0% (Tech Assistant) + 5.0% (Founder)

**Actionable Subtasks:**

  • Implement core data models and database schema.
  • Develop **CRUD API endpoints** for story and asset management.
  • Implement the core business logic (e.g., asset minting and transfer protocols).

**Definition of Done (DoD):** All core API endpoints are functional and tested. Unit test coverage meets $70\%$ minimum requirement. (Total 20.0%)

5. Story Animation / Front-End

Credit: 10.0% (Tech Assistant) + 5.0% (Founder)

**Actionable Subtasks:**

  • Develop user interface (UI) to display an animated story example.
  • Implement front-end functionality to interact with core backend API (e.g., 'View Asset' button).
  • Ensure responsive design across desktop and mobile browsers.

**Definition of Done (DoD):** A functional, fully styled, and responsive front-end demo connects successfully to the live backend service. (Total 15.0%)

6. Tech Support (Founder Empowerment)

Credit: 5.0% (Technical Assistant)

**Actionable Subtasks:**

  • Develop and conduct a training session on using the platform and asset creation tools.
  • Create clear documentation/tutorials for technical onboarding.
  • Provide direct support for trading and platform interactions.

**Definition of Done (DoD):** Founder confirms successful completion of training and is able to independently create, deploy, and trade a test asset using the new documentation. (Total 5.0%)

Phase III: Market & Growth Strategy (35% Weight)

7. Story Generation (Content/IP)

Credit: 10.0% (Founder) + 5.0% (Market Liaison)

**Actionable Subtasks:**

  • Finalize the core IP, narrative, and brand identity of the project.
  • Draft the **Whitepaper/Litepaper** defining project vision and mechanics.
  • Identify three core use cases for the platform's assets.

**Definition of Done (DoD):** Whitepaper/Litepaper is drafted, reviewed, and finalized. Core IP is registered (if applicable) or legally protected. (Total 15.0%)

8. Market Valuation & Pricing

Credit: 10.0% (Market Liaison)

**Actionable Subtasks:**

  • Conduct a competitive analysis of similar platforms/projects.
  • Develop a detailed **pricing model** and initial revenue projections.
  • Define the Target Audience (TA) profile and key communication channels.

**Definition of Done (DoD):** A formal market analysis and pricing strategy document is delivered and approved by the Founder. (Total 10.0%)

9. Founder Recruitment

Credit: 5.0% (Market Liaison) + 5.0% (Founder)

**Actionable Subtasks:**

  • Establish contact and partnership with key LDN university departments/incubators.
  • Develop pitch deck and recruitment materials.
  • Secure commitments from the **first three external project founders/contributors**.

**Definition of Done (DoD):** Three external founders/contributors are legally onboarded (signed NDA/CLA) and have begun their initial task. (Total 10.0%)
