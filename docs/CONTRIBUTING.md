# Contributing - Project Engine

First off, thanks for taking the time to contribute! ❤️

## How can I contribute?
### Reporting Bugs
Before creating a bug report, please check the issues list to see if the problem has already been reported.

### Pull Requests
1. Note in time templates are planned for both Issue reporting and Pull Requests.
1. Fork the repo and create your branch from `main`.
3. If you've added code that should be tested, add tests.
4. Ensure the test suite passes.
5. Use clear commit messages.
